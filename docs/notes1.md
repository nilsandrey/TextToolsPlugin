What's in the plugin now
~80 commands across 8 groups, all accessible from the command palette prefixed with Text Tools:.

Group	Commands
Case / format	Upper, Lower, camelCase, PascalCase, CONSTANT_CASE, dash-case, snake_case, dot.case, Title Case, sPoNgE cAsE, Swap case, Separate with spaces/slashes/backslashes, Reverse chars, Slugify, Latinize
Line operations	Remove duplicates / adjacent duplicates / surplus blanks / blank / empty, Keep only duplicates, Shuffle, Sort (case-sensitive/insensitive × asc/desc, by length, word count, last word), Trim (both/start/end), Remove whitespace, Collapse whitespace, Replace newlines with space, Prefix / Suffix / Wrap lines, Split lines, Join lines / Join every N lines, Count occurrences, Pad start / Pad end
Filter lines	Keep/remove by string or regex × in-place or into new note (8 commands) — with 10-item history
Encoding	URL encode/decode, HTML entity encode/decode, Base64 encode/decode, JSON escape/unescape, Decimal↔Hex
Numbers	Increase/decrease by 1, Increase/decrease by custom step
Generate	4 GUID formats, random int/float/hex from range, Lorem ipsum sentence/paragraph, 7 sequences (letters, NATO, months, days), 3 timestamp formats
Advanced	Format as table (any delimiter), Extract via regex (capture groups + replacement), Duplicate selection
Text slots	Set/paste × 5 persistent slots
Line numbers	Sequential (start at 1) / real file line numbers
